# 第4章：线段的特征序列与划分

## 4.1 线段的严格定义

笔是K线的更高级连接，而**线段**则是笔的更高级连接。
为了过滤掉单纯由“笔”带来的假信号（比如一笔回调看起来很猛，但随即又新高），缠论引入了线段概念。

**基本公式**：
线段 = 至少连续三笔，且前三笔必须有**重叠部分**。

::: chart:segment_basics :::

*   **直观理解**：线段就像是K线图中的“波浪”。一个完整的上涨浪（向上线段），里面至少包含了“上-下-上”三个动作。
*   **方向唯一**：线段和笔一样，也有方向，且必须交替连接。

## 4.2 特征序列：把笔当成K线看

为了科学地判断线段什么时候结束，我们需要把“笔”抽象成一种特殊的“K线”，这就是**特征序列**。

*   **向上线段中**：主要关注**回调的笔**（向下笔）。我们把每一根向下笔，看作一根只有高低点的K线（特征元素）。
*   **向下线段中**：主要关注**反弹的笔**（向上笔）。

::: chart:segment_feature_sequence :::

**为什么这很重要？**
因为判断线段结束，就变成了判断“特征序列K线”是否出现了**顶分型**（对应向上线段结束）或**底分型**（对应向下线段结束）。这实现了逻辑闭环：

*   K线 -> 分型 -> 笔
*   笔（作为特征序列） -> 分型 -> 线段

## 4.3 线段是如何被破坏的？

线段什么时候结束？只有一种情况：**被另一条反向线段破坏**。
但在实际操作中，我们要立刻判断当下的一笔是否引发了破坏，主要通过**特征序列的分型**来确认。这分为两种情况：

### 第一种情况：笔破坏（直接崩塌）

**场景**：向上线段中，突然出现一笔极强的下跌。
**特征**：这根下跌笔（反向笔）极其有力，直接跌穿了前一个特征元素（最后一根向下笔）的底，形成了**包含关系**。
**结论**：一旦出现这种情况，线段大概率直接结束，根本不需要等什么反弹确认。

::: chart:segment_break_case1 :::

### 第二种情况：线段破坏（纠结后死亡）

**场景**：向上线段中，开始回调的一笔力度一般（没跌破前一个特征元素的底），也就是不仅包含。
**后续**：这时候线段还没死。需要看接下来的**反弹**及**再下跌**。如果：

1.  反弹不创新高。
2.  再下跌跌破了第一笔回调的低点。

**结论**：这就构成了特征序列的“顶分型”，线段在此刻确认结束。

## 4.4 学习要点与实战心法

1.  **别被概念绕晕**：初学者最容易死磕“特征序列包含处理”，其实**核心就是看力度**。
    *   回撤力度大（破前低）：警惕，可能直接转折（Case 1）。
    *   回撤力度小（不破前低）：安心持股，等待看反弹情况（Case 2）。
2.  **缺口是关键**：第二种情况破坏，本质上是因为第一笔回撤和前一序列元素之间有“缺口”（没有重合/被包含），所以需要后面的走势来补刀确认。
3.  **操作建议**：用线段来看大方向，用笔来找切入点。不要在1分钟图上纠结线段，会累死；在30分钟或日线级别画线段，方向感会极强。
