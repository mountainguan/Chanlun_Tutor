# 第2章：分型 - 顶分型与底分型

## 2.1 分型的几何意义
在K线经过包含处理变得“干净”之后，我们就可以寻找形态了。分型是缠论中最基础的形态，它像一个钩子，把走势勾住，发生转折。

分型由**三根**经过处理后的K线组成。

## 2.2 顶分型 (Top Fenxing)
**形态**：中间K线最高，左边和右边的K线都比它低。

*   **High(中) > High(左)** 且 **High(中) > High(右)**
*   **Low(中) > Low(左)** 且 **Low(中) > Low(右)**

::: chart:fenxing_top :::

**实战含义**：

*   **短期见顶信号**：如果出现在上涨趋势末端，往往意味着多头力量耗尽，空头开始反击。
*   **心理博弈**：市场冲高回落，第三根K线的走势至关重要。如果第三根K线大幅下跌，跌破第一根K线低点，那是强烈的卖出信号。

## 2.3 底分型 (Bottom Fenxing)
**形态**：中间K线最低，左边和右边的K线都比它高。

*   **High(中) < High(左)** 且 **High(中) < High(右)**
*   **Low(中) < Low(左)** 且 **Low(中) < Low(右)**

::: chart:fenxing_bottom :::

**实战含义**：

*   **短期见底信号**：空头砸不动了，多头开始尝试反攻。
*   **入场时机**：很多短线买点都是在底分型确认形成的瞬间产生的。

## 2.4 分型的力度辨析（实战核心）
分型只是“停顿”，是否能演变为“转折”，关键在于分型的**力度**。判断分型能否成笔，或者能否引发大级别的反转，第三根K线的表现至关重要。

### 1. 顶分型的强弱分类
*   **最强力度（毁灭性打击）**
    第三根K线不仅跌破第二根（中间）的低点，甚至直接**跌破第一根K线的最低点**，形成“断头铡刀”或“穿头破脚”。此时如果带有向下跳空缺口（Gap），杀伤力极大，往往意味着行情的终结。
    ::: chart:fenxing_top_strong :::
    > **案例描述**：股价经过一波急速拉升后，中间K线收出长上影线（避雷针）；次日（第三根）低开低走，收出一根饱满的大阴线，一举吞没前两日的涨幅。这是最强烈的离场信号。

*   **一般力度**
    第三根K线跌破了中间K线的低点，但收盘价仍停留在第一根K线的价格区间内（未跌破第一根低点）。此时市场进入犹豫期，可能演变成震荡，需要观察第四根K线的走向。

*   **最弱力度（中继形态）**
    第三根K线虽然在几何上构成了顶分型（高点降低，低点降低），但实体很小，或者带有长下影线，收盘价甚至没有有效跌破中间K线的实体下沿。
    ::: chart:fenxing_top_weak :::
    > **案例描述**：上涨途中，股价收出一根小阴线，刚好构成顶分型，但并未跌破5日均线。这往往是主力洗盘的动作，被称为“中继顶分型”，随后股价大概率会创新高，破坏该分型。

### 2. 底分型的强弱分类
*   **最强力度（V型反转）**
    第三根K线必须强势，一根大阳线拔地而起，**收盘价直接站上第一根K线的高点**。如果伴随成交量放大，这往往是见底的铁证。
    ::: chart:fenxing_bottom_strong :::

*   **弱势底分型**
    第三根K线虽然站上了中间K线的高点，但留下了长上影线，或者收盘价在这个分型区间的下半部分。这种底分型很脆弱，容易演变成下跌中继（即：跌-盘(弱底分)-跌）。

### 3. 结合心理与环境判断
*   **均线配合**：顶分型出现时，如果K线同时跌破了5日均线，有效性大幅提升。
*   **背驰验证**：如果该分型出现在MACD背驰的位置（比如股价创新高，红柱子面积却在缩小），那么这个顶分型几乎100%是转折点。

**记住口诀**：
> "乱军丛中看分型，第三K线定乾坤。不破前低非真顶，不破前高非真底。"

## 2.5 进阶：分型的“真假”与延续

找到了顶分型，市场就一定下跌吗？不一定。分型只是一种“形态学”上的构造，它仅仅代表了暂时的停顿。在实战中，我们需要分辨分型是否有效（能否引发转折）。

### 1. 中继分型（假信号）
很多时候，趋势非常强烈。上涨途中出现了一个顶分型，但这是一个**中继分型**。

*   **特征**：顶分型出现后，股价并没有有效下跌，或者下跌幅度极小（构不成一笔），随后又马上掉头向上，突破了顶分型的高点。
*   **含义**：这说明刚才的分型只是多头歇了一口气（中继），原来的上涨趋势还在延续。此前的顶分型宣告**失效**。

### 2. 分型的破坏
一旦新的K线收盘价突破了顶分型的高点（或跌破底分型的低点），原来的分型就被**破坏**了，我们需要重新在这个新位置寻找新的分型。永远跟随当下的最新分型。

### 3. 更高级别的分型（预告）
我们在本章讨论的是由**K线**构成的分型。
如果我们把视角拉高，把“一笔”看作是一根K线，那么“笔”和“笔”之间也会构成分型结构。

*   这种由笔构成的分型，被称为**线段的特征序列分型**。
*   它决定了比“笔”更高一级的走势——**线段**的结束。
